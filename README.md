# TeutonicTerror Portfolio Website

![Website Preview](images/profilewithoutBG.png)  
**Live Site**: [teutonicterror.pages.dev](https://teutonicterror.pages.dev)

A professional portfolio website showcasing development projects, technical blog posts, and contact information.

## Key Features

- **Modern Dark Theme**: Sleek UI with gradient effects
- **Interactive Elements**:
  - Dynamic header background effects
  - Smooth hover animations
  - Responsive project cards
- **Multi-Page Navigation**:
  - Project showcase
  - Development blog
  - Direct contact methods

## Page Structure

| Page          | Description                          | File          |
|---------------|--------------------------------------|---------------|
| Home          | Main portfolio landing page          | `index.html`  |
| Blog          | Technical articles and updates       | `blogs.html`  |
| Contact       | Direct communication channels        | `contacts.html` |
